    Über GitList

    GitList ist eine kostenlose Open-Source-Anwendung, die es Dir ermöglicht, Link-Sammlungen zu interessanten Videos, Podcasts oder Webseiten übersichtlich zusammenzustellen. Zur besseren Strukturierung kannst Du Links in Listen und Sammlungen organisieren.

    Zum aktuellen Zeitpunkt befindet sich GitList in einem frühen Entwicklungsstadium. Durch deine Mitarbeit kannst du jedoch jetzt schon zum Erfolg des Projektes beitragen.

    Unterstütze GitList aktiv, indem du Fehler auf GitHub meldest, Verbesserungsvorschläge einreichst oder zur Weiterentwicklung des Projekts beiträgst. Falls du neu bei GitHub bist, findest du hier eine hilfreiche Anleitung für Einsteiger.
